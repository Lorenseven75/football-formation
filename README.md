# Formazione interattiva per Notion

## 1. Personalizzazione
Apri `team.json` e modifica:
- `name`: nome squadra
- `formation`: modulo
- `logo`: URL pubblico dello stemma
- `players`: nome, posizione, ruolo, coordinate, foto e link Notion
- `staff`: allenatore e staff

Ruoli accettati:
- `goalkeeper`
- `defender`
- `midfielder`
- `forward`
- `coach`
- `staff`

Le coordinate `x` e `y` sono percentuali del campo.

## 2. Pubblicazione gratuita
Puoi pubblicare la cartella su:
- GitHub Pages
- Netlify Drop
- Vercel

Con Netlify Drop:
1. estrai lo ZIP;
2. trascina la cartella sul pannello di Netlify Drop;
3. copia il link pubblico ottenuto.

## 3. Inserimento in Notion
1. apri la pagina della squadra;
2. digita `/embed`;
3. incolla il link pubblico;
4. allarga il blocco alla larghezza della pagina.

## Nota sui link Notion
Perché i collegamenti funzionino per altri lettori, le pagine dei personaggi devono essere condivise con loro oppure pubblicate sul web.
